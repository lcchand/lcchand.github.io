Colophon

Semantic HTML5 tags are used in my markup to describe the structure of the document and improve the accessibility of my site. WAI-ARIA landmark rules have been added as attributes to the relevant element or elements. These rules describe the elements they're attached to, and may even add functionality that wouldn't be there otherwise. Alternative text is used wherever any image elements are included.

Presentation

The presentation layer of this site was coded using the CSS preprocessor (SCCS) Sass. Related sections have been broke into partials. Sass Variables are used for repetitive values as well as CSS Variables (custom properites). Normalize.css was included at the start of the style sheet to make browsers render all elements more consistently. To keep the code modular and easier to understand, I used BEM methodology for naming classes. CSS animation is used throughout the site in the mobile menu, hero and category sections of the portfolio main pages.

Workflow

Gulp in conjunction with the Node Ecosystem was the toolkit I choose to automate tasks in my development workflow. Compiling the Sass was executed using Node Sass to generate the CSS3 stylesheet. Live browser updating in Chrome, Firefox and Safari was accomplished with BrowserSync. Everyone has their favorite text editor. Mine happens to be Emacs running in MacOSX. The packages (Emacs for plugins) used were Emmet, Web, Sass and Company-mode for code completion. JS2-mode and Tern were used for Javascript. Live code linting was peformed with FlyCheck. Eshell was used within an Emacs buffer window for monitoring Gulp output messages. Git repository maintenance was executed using the command line in a terminal window.

Interactivity

To enhance navigating the site and the previewing projects was executed using Javascript. Mobilenav.js adds functionality to the hamburger menu using Eventlisteners and class manipulation. Modal.js creates a Modal Overlay for selected Portfolio items using EventListeners, DOM manipulation and Data Attributes. The Intersection Observer API is used for triggering transitions between sections and highlighting navigation.
